# Jobify

An app built for individuals looking for developers or jobs across the globoe.

![Jobify](./jobify.png)


## [Live version](https://jobify-7k8g8ix0y.vercel.app/)

## Technologies

- React
- React Hooks
- Tailwindcss

## Install

```sh
  git git@github.com:abruzy/jobify.git
  cd ~/jobify
  run yarn install in your terminal
  ```

  ## Usage

  run 

  ```sh
  yarn start
  ```

```sh
yarn style:watch
```
